# WorkSpace Manager (WSM)

WorkSpace Manager (WSM) is a collection of Ansible roles to help manage your personal workspace.

## Installation

1. Install Ansible

2. Clone WSM to your home directory

    ```bash
    git clone https://github.com/trly/wsm.git ~/.wsm
    ```

## Usage

```bash
cd ~/.wsm

./install.sh
```
